雨课堂导出工具

雨课堂试卷一键导出，支持小窗试卷和无图试卷，导出为 Markdown / JSON，方便复习与整理题库。

查看 GitHub 仓库 下载源码 安装与使用说明
功能概览

一键导出雨课堂试卷

  • 支持导出试卷题目、选项和答题记录
  • 自动生成 Markdown / JSON 文件
  • 适合复习、整理错题与搭建题库
页面模式

小窗试卷 & 无图试卷

  • 小窗试卷（iframe）：适配学生答题小窗
  • 无图试卷（正常试卷）：适配成绩/解析页面
  • 自动识别当前页面类型，也可手动切换
导出格式

Markdown / JSON

  • Markdown：适合人类阅读与打印
  • JSON：适合程序处理与二次开发
  • 文件名包含试卷标题与 exam_id，便于归档
安装方式

扩展 & 用户脚本

  • Chrome / Edge：开发者模式加载 extension/
  • Firefox：临时载入 extension/manifest.json
  • 用户脚本：在 Tampermonkey 粘贴 userscripts 中脚本
隐私与安全

本地处理 · 不上传

  • 所有操作在本地浏览器完成，不上传数据
  • 最小权限：仅使用 storage 与目标站点 host 权限
  • 仅导出你本来就能在页面上看到的内容
